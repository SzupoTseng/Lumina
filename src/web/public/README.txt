Drop a VRM file named "avatar.vrm" into this folder, then reload the dev server.

Free placeholder options:
- VRoid Studio (https://vroid.com/studio) — capture → export VRM
- Sample VRMs: https://github.com/pixiv/three-vrm/tree/dev/packages/three-vrm/examples/models

If no avatar.vrm is present, the scene falls back to a capsule + sphere stand-in.
